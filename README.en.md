# TeamFlow

Production-ready multi-tenant SaaS collaboration platform built with Next.js, NestJS, PostgreSQL, Docker, and AWS.

🇯🇵 Japanese README: [README.md](./README.md)

---

# Overview

TeamFlow is a scalable full-stack SaaS platform designed for team collaboration, task management, and workspace-based project organization.

This project was built to simulate a production-grade cloud-native SaaS architecture using modern enterprise development practices.

---

# Tech Stack

## Frontend

- Next.js (App Router)
- TypeScript
- TailwindCSS
- React Query
- Zustand
- shadcn/ui

## Backend

- NestJS
- PostgreSQL
- Prisma ORM
- JWT Authentication
- Zod Validation

## Infrastructure

- Docker
- GitHub Actions
- AWS ECS/Fargate
- Amazon RDS
- Amazon S3
- CloudFront

---

# Key Features

- Multi-tenant workspace architecture
- RBAC authorization system
- JWT authentication
- Dockerized development environment
- Swagger API documentation
- Scalable monorepo structure
